# Библиотека `<utility>` в C++: примеры использования `std::pair`

Заголовочный файл `<utility>` содержит вспомогательные компоненты, наиболее известный из которых — **`std::pair`**. Это шаблонный класс, хранящий два разнотипных значения как единый объект. Часто применяется для возврата двух значений из функции, хранения пар ключ-значение в ассоциативных контейнерах (например, `std::map`) и в других ситуациях.

## Основные возможности `std::pair`

- Создание пары: `std::pair<T1, T2> p(value1, value2);`
- Доступ к элементам: `p.first`, `p.second`
- Удобное создание с выводом типов: `std::make_pair(value1, value2)` (C++11) или использование классовых шаблонных аргументов (CTAD) с C++17.
- Операции сравнения (`==`, `!=`, `<`, `<=`, `>`, `>=`) — лексикографическое сравнение.
- Функция `std::swap` для двух объектов `pair`.

## Примеры использования

### 1. Без функции (просто работа с парой в `main`)

```cpp
#include <iostream>
#include <utility>
#include <string>

int main() {
    std::pair<int, std::string> person(25, "Alice");
    std::cout << "Age: " << person.first << ", Name: " << person.second << '\n';

    auto point = std::make_pair(3.5, 2.1);
    std::cout << "x = " << point.first << ", y = " << point.second << '\n';

    std::pair<int, int> a(1, 2), b(1, 3);
    if (a < b)
        std::cout << "a меньше b\n";

    return 0;
}
